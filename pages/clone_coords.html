CCH1 Coordinate Cloning Interface



As of November 2018, the CCH1 Coordinate Cloning interface has been retired. The CCH2 symbiota portal contains the tools for adding new georeferences to specimens of CCH member herbaria.


To query the archived version of the CCH1 cloning interface, please use this link:
http://ucjeps.berkeley.edu/consortium/clone_coords_2017.html


NOTA BENE: This link is provided for archival use only. The georeference data in CCH1, which contains the California records found in CCH2, can be searched. However, no new coordinates can be added to the CCH1 georeference buffer.
